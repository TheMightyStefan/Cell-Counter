The implementation of erosion, fundamental morphological operation. 

The dilation of a binary image F by a structuring element S produces a new binary image G with ones in all locations of a structuring element\textquotesingle{}s origin at which that structuring element S hits the input image F, i.\+e. g(x,y) = 1 if S hits F and 0 otherwise, repeating for all pixel coordinates (x,y).
